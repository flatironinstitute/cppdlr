\doxysubsection{Detailed Description}
Class responsible for all DLR imaginary frequency operations, including building imaginary frequency grid and transformations. 

\begin{DoxyNote}{Note}
First dimension of all Green\textquotesingle{}s function and coefficient arrays must be DLR rank r. 
\end{DoxyNote}
